# 思源笔记日历面板插件

简体中文 \| [English](https://github.com/51junjie/siyuan-plugin-calendar-multi/blob/main/README_en_US.md)

一个思源笔记日历面板插件，可以快速创建日记与周记。

![项目截图](https://imgs.pprain.space/2026/01/41d3d243f69952db7cf91a75d4963246.png)

## 主要功能

- 日历功能
  - 可选按钮入口位置
  - 可选显示周号
  - 可选每周开始日(周日/周一)
  - 边栏模式下宽度缩放自适应
- 日记周记功能
  - 点击日期可快速创建或跳转到该日期日记
  - 可选开启周记功能，开启后点击周号可创建相应周周记
  - 支持设置周记的创建位置与模板
  - 支持多个日历快速切换（最多三个）
